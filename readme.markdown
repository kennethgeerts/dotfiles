# Kenneth's dotfiles

## Install

```zsh
curl -fsSL https://raw.githubusercontent.com/kennethgeerts/dotfiles/HEAD/install | zsh
```

## Omarchy

These dotfiles are for MacOS. For linux, just install Omarchy.
